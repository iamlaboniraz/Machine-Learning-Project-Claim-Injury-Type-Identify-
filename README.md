# Introduction
This project is a classification problem with the goal of predicting the Claim Injury Type for workers' compensation claims, using data from 2020 to 2022, to automate decision-making and improve WCB processes. In this project, we took an in-depth analysis of the dataset with a focus on key attributes such as injury details, claimant
demographics, and workplace information, which provided valuable insights into the patterns and relationships within the data. First, we tried to carefully clean and preprocess to prepare the whole dataset so that in our modelling and exploratory data analysis part, it can be performed better. Then we built and applied several machine learning models using advanced classification techniques, fine-tuning and optimizing them to maximize their performance. In this process, we developed a model capable of accurately predicting injury types, reducing
manual intervention in the claims review process. In this report, we will discuss the steps and the results of our analysis that we took to complete this project.
